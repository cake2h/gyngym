# GymGym — Документация продукта

## Обзор

**GymGym** — мобильное веб-приложение для отслеживания фитнес-прогресса. Замена Excel-таблиц для тренировок, замеров и личных заметок.

---

## 1. Технический стек

### Рекомендация: Laravel + React SPA

| Слой | Технология | Обоснование |
|------|------------|-------------|
| **PHP** | 8.2+ (рекомендуется 8.3) | Требование Laravel 12 |
| **Backend** | Laravel 12 | API, auth, БД. Отлично работает на shared hosting (reg.ru). Готов к Stripe, очередям, AI-интеграциям |
| **Frontend** | React 19 + Vite 6 | SPA, mobile-first. Сборка → статические файлы в `public/` Laravel |
| **БД** | MySQL 8 | Стандарт для shared hosting |
| **PWA** | vite-plugin-pwa | Офлайн, добавление на главный экран |
| **UI** | Tailwind CSS 4 | Быстрая вёрстка, адаптивность |

> **Важно:** Laravel 12 требует PHP 8.2+. Проверьте версию: `php -v`

### Почему не Node.js backend?

Shared hosting reg.ru обычно не поддерживает Node.js. Laravel + React (собранный в статику) — один PHP-хостинг, один домен, всё работает.

### Альтернатива на будущее

При переходе на VPS можно рассмотреть: Laravel + Inertia.js + React (меньше дублирования) или отдельный Node.js API. Сейчас — Laravel API + React SPA оптимальны.

## 3. Функционал

### 3.1 Личный кабинет и профиль (MVP)

#### Регистрация
1. **Шаг 1**: Ввод email
2. **Шаг 2**: Поля — ФИО, Возраст, Вес, Пол
3. Подтверждение почты — **не требуется** (на первом этапе)

#### Профиль
- Доступ: крайняя правая кнопка в нижнем меню
- Редактирование: ФИО, возраст, вес, пол

#### Замеры
- **Мужчины**: плечи, грудь, запястья, предплечья, пресс
- **Женщины**: плечи, грудь, попа, талия
- Ввод: значение + дата
- Визуализация: графики по каждому параметру (линейные/по времени)

### 3.2 Дневник тренировок (MVP)

#### Модель данных
- **План**: 2 тренировки в неделю (например, Пн и Чт)
- **Структура**: Тренировка A и Тренировка B — одинаковый набор упражнений каждую неделю
- **Прогресс**: пользователь вводит вес и/или количество повторений на каждую тренировку

#### Интерфейс
- Календарь с отметками тренировок
- Таблицы: упражнение | подходы | вес | повторения
- Возможность добавлять упражнения в тренировку из базы
- История по неделям для сравнения

#### База упражнений
- Предзаполненный список популярных упражнений
- Категории: грудь, спина, ноги, плечи, руки, пресс

### 3.3 База знаний (MVP)

- **Приватная** для каждого пользователя
- Типы записей: текст, ссылки, изображения, видео
- Связь с упражнениями (опционально)
- Поиск и теги (позже)

## 6. API Endpoints (черновик)

### Auth
- `POST /api/register` — регистрация (email → данные)
- `POST /api/login` — вход
- `POST /api/logout` — выход

### Profile
- `GET/PUT /api/profile` — профиль
- `GET/POST /api/measurements` — замеры
- `GET /api/measurements/chart/{type}` — данные для графика

### Training
- `GET /api/calendar` — календарь тренировок
- `GET /api/workouts` — список тренировок
- `GET /api/exercises` — база упражнений
- `POST /api/workout-sessions` — запись выполненной тренировки

### Knowledge Base
- `CRUD /api/notes` — заметки
