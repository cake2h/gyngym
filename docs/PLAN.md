# План разработки GymGym

## Фазы и этапы

---

## Фаза 0: Подготовка (1–2 дня)

- [x] Инициализация Laravel 12 проекта
- [x] Настройка React 19 + Vite 7 в Laravel
- [ ] Настройка MySQL (или SQLite для разработки)
- [x] Базовая структура API (Sanctum для auth)
- [x] PWA манифест и Service Worker

---

## Фаза 1: Аутентификация и профиль (3–5 дней)

### 1.1 Регистрация
- [x] API: `POST /api/register` — шаг 1 (email)
- [x] API: `POST /api/register/complete` — шаг 2 (ФИО, возраст, вес, пол, пароль)
- [x] React: экраны регистрации (2 шага)
- [x] React: экран входа
- [x] Laravel Sanctum: токены для SPA

### 1.2 Профиль
- [x] Модель User + миграция (age, weight, gender)
- [x] API: GET/PUT профиля
- [x] React: страница профиля, форма редактирования
- [x] Нижнее меню: 4 вкладки, профиль — крайняя правая

### 1.3 Замеры
- [x] Модель Measurement + миграция (user_id, type, value, measured_at)
- [x] Типы замеров: мужские/женские
- [x] API: CRUD замеров
- [x] API: данные для графиков (по типу)
- [x] React: форма ввода замера
- [x] React: графики (Recharts)

---

## Фаза 2: Дневник тренировок (5–7 дней)

### 2.1 Модели данных
- [ ] Exercise — база упражнений (название, категория, описание)
- [ ] WorkoutTemplate — шаблон тренировки (название, список упражнений)
- [ ] WorkoutSession — выполненная тренировка (дата, шаблон)
- [ ] WorkoutSet — подход (упражнение, вес, повторения, порядок)
- [ ] Сидер: 50+ популярных упражнений

### 2.2 API
- [ ] GET /api/exercises — список упражнений (с фильтрами)
- [ ] GET /api/workout-templates — шаблоны пользователя (A, B)
- [ ] CRUD шаблонов и сессий
- [ ] GET /api/calendar?month= — тренировки по месяцам

### 2.3 Интерфейс
- [ ] Календарь (библиотека или свой компонент)
- [ ] Страница тренировки: таблица упражнений + ввод веса/повторений
- [ ] Создание/редактирование шаблона (добавление упражнений)
- [ ] История: сравнение недель
